Início Rápido

  1. Instale e Abra: Conceda permissões de câmera, microfone e localização quando solicitado.
  2. Monte o Celular: Fixe com segurança no para-brisa em orientação paisagem, com a câmera traseira voltada para frente.
  3. Ligue o Dispositivo: Conecte a uma fonte de energia que ligue com a ignição do carro (ACC). O app começa a gravar automaticamente ao detectar energia e para quando a energia é desligada.
  4. Armazenamento: Os vídeos são salvos internamente por padrão. Você pode mudar para cartão SD ou armazenamento externo nas configurações.
  5. Uploads (Opcional): Configure uploads para seu computador, NAS ou nuvem (Google Drive, OneDrive) via Wi-Fi.

Introdução

O DashCam transforma um Android antigo em uma câmera veicular confiável e completa. Projetado para estabilidade, automação e simplicidade, grava automaticamente quando o carro liga e salva ou faz upload das gravações ao desligar.

Principais Recursos

  • Gravação automática vinculada à ignição ou impacto
  • Detecção de impacto com buffer circular de vídeo
  • Registro de dados de GPS, velocidade e sensores de movimento
  • Uploads seguros para compartilhamentos de rede (SMB/CIFS) ou nuvem (Google Drive, OneDrive)
  • Múltiplos formatos de saída: MP4, SRT, CSV, GPX
  • Armazenamento de credenciais criptografado via Android Keystore
  • Resolução de hostname via NetBIOS para redes Windows
  • Otimizado para mínima interação — instale uma vez, use diariamente

Primeiros Passos

Permissões

Conceda estas permissões no primeiro uso:

  • Câmera – Necessária para gravação de vídeo.
  • Microfone – Para captura de áudio.
  • Localização – Para registro de GPS e velocidade.
  • Armazenamento – (Android 9 ou inferior) Para salvar gravações.

Montagem do Celular

  • Use um suporte de para-brisa ou painel com visão estável para frente.
  • Mantenha os cabos organizados e fora do campo de visão.
  • Conecte a energia de um carregador ligado à ignição (ACC) para operação automática.

Comportamento de Energia

  • Ignição Ligada (Energia Conectada): Gravação inicia automaticamente.
  • Ignição Desligada (Energia Desconectada): Gravação para com segurança e uploads começam (se configurado). O celular permanece acordado para finalizar uploads e depois entra em modo de economia de energia.

Tela Principal

  • Canto Superior Esquerdo: Data e hora
  • Canto Inferior Esquerdo: Coordenadas GPS e direção
  • Canto Inferior Direito: Velocidade (mph ou km/h)
  • Centro: Pré-visualização da câmera e indicador de gravação
  • Botão Flutuante: Toque no vermelho para iniciar, branco para parar gravação manual

Aviso: Se aparecer "EMULADOR – Gravação de vídeo desativada", você está em um dispositivo virtual. Use um celular real para gravar.

Modos de Gravação

1. AUTO (Detecção de Impacto)

Grava continuamente em pequenos loops, salvando apenas os últimos dois minutos. Quando um impacto (1.5G–3.0G) é detectado, o DashCam preserva:

  • Dois minutos antes do impacto
  • Dez segundos após
  • Todos os dados de telemetria (GPS + sensores)

2. POR VIAGEM

Inicia quando a ignição liga, para quando desliga. Salva cada viagem como um arquivo único com logs sincronizados.

3. Loop Temporizado (5/10/30 minutos)

Divide gravações em arquivos rotativos menores para facilitar upload e gerenciamento.

4. Manual

Toque no botão vermelho para iniciar ou parar a gravação a qualquer momento.

Visão Geral das Configurações

Exibição

  • Unidades de Velocidade: mph ou km/h
  • Qualidade de Vídeo: 720p, 1080p (padrão), 4K
  • Formato de Data/Hora: Personalização regional
  • Manter Tela Ligada: Evita desligamento durante gravação

Gravação

  • Modo: AUTO, POR VIAGEM, 5/10/30 min
  • Sensibilidade de Impacto: Baixa (3.0G), Média (2.0G padrão), Alta (1.5G)

Gerenciamento de Armazenamento

Local Padrão

/Android/data/com.dashcam.app/files/DashCam/

Armazenamento Externo ou Cartão SD

  1. Vá em Configurações → Armazenamento.
  2. Escolha EXTERNO.
  3. Toque em Selecionar Pasta Personalizada e escolha o caminho do SD ou USB.

Tamanhos de Arquivo (Aproximado)

Qualidade MB/min Uso Recomendado
720p 50 Viagens longas, pouco espaço
1080p 80 Qualidade equilibrada (padrão)
4K 200 Alta definição, muito espaço

Dica: Logs de telemetria (.srt, .csv, .gpx) ficam abaixo de 0,2 MB/min juntos.

Configuração de Upload

O DashCam suporta uploads automáticos via Wi-Fi para compartilhamentos SMB/CIFS, Google Drive ou OneDrive.

Configuração SMB

  1. Ative Uploads nas Configurações.
  2. Defina o Momento do Upload: Manual, Durante Gravação ou Após Gravação.
  3. Política de Rede: Apenas Wi-Fi (padrão) ou Permitir Dados.
  4. Insira os Detalhes do Servidor:
    • Host: Nome do computador (NetBIOS) ou IP.
    • Compartilhamento: Nome da pasta, ex: dashcam.
    • Usuário / Senha: Credenciais de conta local do Windows.

Resolução de Nome (Redes Windows)

O DashCam suporta resolução NetBIOS, permitindo usar nomes de computadores Windows em vez de IPs (ex: MEUPC). Certifique-se de que ambos os dispositivos estejam na mesma rede Wi-Fi e que a Descoberta de Rede esteja ativada no Windows.

Usuários de macOS ou Linux podem precisar conectar usando o IP.

Por que usar uma Conta Local do Windows

O Windows 10+ não suporta mais compartilhamentos de convidado. Crie um usuário local dedicado com senha:

  1. Win + I → Contas → Família e outros usuários.
  2. Selecione Adicionar conta → Adicionar usuário sem conta Microsoft.
  3. Crie o nome de usuário (ex: dashcam_user) e senha.
  4. Dê permissões de Leitura/Escrita nas abas Compartilhamento e Segurança.

Dica: Descubra o IP do PC pelo Prompt de Comando → ipconfig → Endereço IPv4.

Uploads em Nuvem

O DashCam também pode fazer upload direto para a nuvem.

  • Google Drive: Totalmente suportado. Autorize sua conta Google uma vez nas configurações.
  • OneDrive: Totalmente suportado. Faça login na conta Microsoft e selecione a pasta de destino.
  • Dropbox: Planejado para versões futuras se houver demanda.

Uploads são retomados automaticamente se interrompidos, e você pode optar por apagar arquivos locais após upload bem-sucedido.

Resolução de Problemas

Problema Solução Possível
Sem vídeo Conceda permissão de câmera; reinicie o app.
GPS travado Vá para área externa; certifique-se de que o GPS está ativado.
Impacto não detectado Ajuste a sensibilidade; fixe melhor o suporte.
Falha no upload Confirme credenciais; mesma rede Wi-Fi; habilite porta SMB 445.
Nome do host não encontrado Verifique descoberta NetBIOS; use IP como alternativa.
Armazenamento cheio Reduza a qualidade do vídeo ou ative autoexclusão após upload.

Dicas de Desempenho

  • Desative a otimização de bateria do Android para o DashCam.
  • Use um carregador de qualidade (2A+).
  • Evite superaquecimento — luz solar direta pode desligar o aparelho.

Notas Técnicas

  • Vídeo: H.264/AVC MP4 @ 30 fps
  • Telemetria: SRT (GPS/velocidade), CSV (10 Hz sensores), GPX (trajeto)
  • Buffer de Impacto: Loop de dois minutos, salva +10s após impacto
  • Permissões: CAMERA, RECORD_AUDIO, ACCESS_FINE_LOCATION, INTERNET, WAKE_LOCK, FOREGROUND_SERVICE
  • Armazenamento de Credenciais: Criptografado via Android Keystore

Limitações Conhecidas (Beta)

  • Integração com Dropbox pendente de demanda dos usuários.
  • Viagens muito longas podem exigir grande capacidade de armazenamento.

Segurança & Responsabilidade

O DashCam auxilia motoristas — não substitui a direção segura. Sempre siga as leis locais sobre gravação veicular.
