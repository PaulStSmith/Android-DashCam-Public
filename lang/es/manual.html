Inicio Rápido

  1. Instala y Abre: Concede permisos de cámara, micrófono y ubicación cuando se te solicite.
  2. Monta el Teléfono: Fija el dispositivo de forma segura en el parabrisas en orientación horizontal, con la cámara trasera hacia adelante.
  3. Conéctalo: Conecta a una fuente de energía que se active con el encendido del coche (ACC). La app comienza a grabar automáticamente al detectar energía y se detiene cuando se pierde la alimentación.
  4. Almacenamiento: Los vídeos se guardan internamente por defecto. Puedes cambiar a tarjeta SD o almacenamiento externo en los ajustes.
  5. Subidas (Opcional): Configura subidas a tu ordenador, NAS o nube (Google Drive, OneDrive) vía Wi-Fi.

Introducción

DashCam convierte un teléfono Android viejo en una cámara de vehículo confiable y con todas las funciones. Diseñado para estabilidad, automatización y simplicidad, graba automáticamente cuando tu coche arranca y guarda o sube tus grabaciones cuando se detiene.

Características Principales

  • Grabación automática vinculada al encendido o impacto
  • Detección de impactos con búfer circular de video
  • Registro de datos de GPS, velocidad y sensores de movimiento
  • Subidas seguras a recursos compartidos de red (SMB/CIFS) o nube (Google Drive, OneDrive)
  • Múltiples formatos de salida: MP4, SRT, CSV, GPX
  • Almacenamiento de credenciales cifrado mediante Android Keystore
  • Resolución de nombre de host vía NetBIOS para redes Windows
  • Optimizado para mínima interacción del usuario — instala una vez, conduce a diario

Primeros Pasos

Permisos

Concede estos permisos en el primer inicio:

  • Cámara – Requerido para grabación de video.
  • Micrófono – Para captura de audio.
  • Ubicación – Para registro de GPS y velocidad.
  • Almacenamiento – (Android 9 e inferior) Para guardar grabaciones.

Montaje del Teléfono

  • Usa un soporte para parabrisas o salpicadero con una vista estable hacia adelante.
  • Mantén los cables ordenados y fuera de tu línea de visión.
  • Conecta la alimentación desde un cargador conectado al encendido (ACC) para operación automática.

Comportamiento de Energía

  • Encendido Encendido (Energía Conectada): La grabación comienza automáticamente.
  • Encendido Apagado (Energía Perdida): La grabación se detiene de forma segura y comienzan las subidas (si está configurado). El teléfono permanece despierto para finalizar las subidas, luego se duerme para conservar energía.

Pantalla Principal

  • Arriba-Izquierda: Fecha y hora
  • Abajo-Izquierda: Coordenadas GPS y dirección
  • Abajo-Derecha: Velocidad (mph o km/h)
  • Centro: Vista previa de cámara e indicador de grabación
  • Botón Flotante: Toca rojo para iniciar, blanco para detener grabación manual

Nota: Si ves "EMULATOR – Video recording disabled", estás ejecutando en un dispositivo virtual. Usa un teléfono real para grabar.

Modos de Grabación

1. AUTO (Detección de Impactos)

Graba continuamente en bucles cortos, guardando solo los últimos dos minutos. Cuando se detecta un impacto (1.5G–3.0G), DashCam preserva:

  • Dos minutos antes del impacto
  • Diez segundos después
  • Toda la telemetría (GPS + registros de sensores)

2. PER_TRIP

Comienza cuando se enciende el encendido, se detiene cuando se apaga. Guarda cada viaje como un único archivo con registros sincronizados.

3. Bucle Temporizado (5/10/30 minutos)

Divide las grabaciones en archivos más pequeños rotatorios para facilitar la subida y gestión.

4. Manual

Toca el botón rojo para iniciar o detener la grabación en cualquier momento.

Resumen de Ajustes

Pantalla

  • Unidades de Velocidad: mph o km/h
  • Calidad de Video: 720p, 1080p (predeterminado), 4K
  • Formato de Fecha/Hora: Personalización regional
  • Mantener Pantalla Encendida: Evitar tiempo de espera durante la grabación

Grabación

  • Modo: AUTO, PER_TRIP, 5/10/30 min
  • Sensibilidad de Impacto: Baja (3.0G), Media (2.0G predeterminado), Alta (1.5G)

Gestión de Almacenamiento

Ubicación Predeterminada

/Android/data/com.dashcam.app/files/DashCam/

Almacenamiento Externo o Tarjeta SD

  1. Ve a Ajustes → Ajustes de Almacenamiento.
  2. Elige EXTERNO.
  3. Toca Seleccionar Carpeta Personalizada y elige tu tarjeta SD o ruta USB.

Tamaños de Archivo (Aproximados)

Calidad MB/min Uso Recomendado
720p 50 Viajes largos, almacenamiento bajo
1080p 80 Calidad equilibrada (predeterminado)
4K 200 Alto detalle, almacenamiento grande

Consejo: Los registros de telemetría (.srt, .csv, .gpx) permanecen bajo 0.2 MB/min combinados.

Configuración de Subidas

DashCam admite subidas automáticas por Wi-Fi a recursos compartidos SMB/CIFS, Google Drive o OneDrive.

Configuración SMB

  1. Habilita Subidas en Ajustes.
  2. Establece Momento de Subida: Manual, Durante Grabación o Después de Grabación.
  3. Política de Red: Solo Wi-Fi (predeterminado) o Permitir Medida.
  4. Introduce Detalles del Servidor:
    • Host: Nombre de host de Windows (NetBIOS) o dirección IP.
    • Recurso Compartido: Nombre del recurso compartido de carpeta, ej., dashcam.
    • Usuario / Contraseña: Credenciales de cuenta local de Windows.

Resolución de Nombre de Host (Redes Windows)

DashCam admite resolución de nombres NetBIOS, permitiéndote usar nombres de host de Windows en lugar de direcciones IP (ej., MYPC). Asegúrate de que ambos dispositivos estén en la misma red Wi-Fi y que Network Discovery esté habilitado en Windows.

Los usuarios de macOS o Linux pueden necesitar conectarse usando la dirección IP.

Por Qué Necesitas una Cuenta Local de Windows

Windows 10+ ya no admite recursos compartidos de invitados. Crea un usuario local dedicado con acceso por contraseña:

  1. Win + I → Cuentas → Familia y otros usuarios.
  2. Selecciona Agregar cuenta → Agregar un usuario sin cuenta de Microsoft.
  3. Crea nombre de usuario (ej., dashcam_user) y contraseña.
  4. Concede permisos de carpeta Lectura/Escritura en las pestañas Compartir y Seguridad.

Consejo: Encuentra la IP de tu PC vía Símbolo del sistema → ipconfig → Dirección IPv4.

Subidas a la Nube

DashCam también puede subir directamente al almacenamiento en la nube.

  • Google Drive: Totalmente compatible. Autoriza tu cuenta de Google una vez en ajustes.
  • OneDrive: Totalmente compatible. Inicia sesión en tu cuenta de Microsoft y selecciona la carpeta de destino.
  • Dropbox: Planificado para versiones futuras si los usuarios lo solicitan.

Las subidas se reintentan automáticamente si se interrumpen, y puedes elegir eliminar archivos locales después de una transferencia exitosa.

Solución de Problemas

Problema Posible Solución
Sin video Concede permiso de cámara; reinicia la app.
GPS atascado Muévete al exterior para señal; asegura que GPS esté activado.
Impacto no detectado Ajusta sensibilidad; asegura el soporte.
Falla la subida Confirma credenciales; asegura misma Wi-Fi; habilita puerto SMB 445.
Nombre de host no encontrado Verifica descubrimiento NetBIOS; usa IP como alternativa.
Almacenamiento lleno Baja calidad de video o habilita eliminación automática después de subir.

Consejos de Rendimiento

  • Deshabilita la optimización de batería de Android para DashCam.
  • Usa un cargador de calidad (2A+).
  • Evita sobrecalentamiento — la luz solar directa puede provocar apagado.

Notas Técnicas

  • Video: H.264/AVC MP4 @ 30 fps
  • Telemetría: SRT (GPS/velocidad), CSV (datos de sensor a 10 Hz), GPX (pista de mapa)
  • Búfer de Impacto: Bucle rotatorio de dos minutos, guarda +10s post-impacto
  • Permisos: CAMERA, RECORD_AUDIO, ACCESS_FINE_LOCATION, INTERNET, WAKE_LOCK, FOREGROUND_SERVICE
  • Almacenamiento de Credenciales: Cifrado vía Android Keystore

Limitaciones Conocidas (Beta)

  • Integración con Dropbox pendiente de demanda de usuarios.
  • Viajes continuos extremadamente largos pueden requerir gran capacidad de almacenamiento.

Seguridad y Responsabilidad

DashCam asiste a los conductores — no es un sustituto de la conducción segura. Sigue siempre las leyes locales sobre grabación en el vehículo.
