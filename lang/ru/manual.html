Быстрый старт

  1. Установка и запуск: Предоставьте разрешения на камеру, микрофон и геолокацию при запросе.
  2. Установка телефона: Надежно закрепите устройство на лобовом стекле в горизонтальном положении, чтобы задняя камера была направлена вперед.
  3. Подключение питания: Подключите к источнику питания, который включается с зажиганием автомобиля (ACC). Приложение автоматически начинает запись при появлении питания и останавливает при его отключении.
  4. Хранение: Видео по умолчанию сохраняются во внутренней памяти. В настройках можно выбрать SD-карту или внешний накопитель.
  5. Загрузка (опционально): Настройте загрузку на ваш домашний компьютер, NAS или облако (Google Drive, OneDrive) через Wi-Fi.

Введение

DashCam превращает старый Android-телефон в надежный, полнофункциональный видеорегистратор для автомобиля. Разработанный для стабильности, автоматизации и простоты, он автоматически записывает, когда ваш автомобиль заводится, и сохраняет или загружает записи, когда он останавливается.

Основные возможности

  • Автоматическая запись, связанная с зажиганием или ударом
  • Обнаружение ударов с круговым видеобуфером
  • Запись данных GPS, скорости и датчиков движения
  • Безопасная загрузка в сетевые папки (SMB/CIFS) или облако (Google Drive, OneDrive)
  • Множество форматов вывода: MP4, SRT, CSV, GPX
  • Зашифрованное хранение учетных данных через Android Keystore
  • Разрешение имен хостов через NetBIOS для сетей Windows
  • Оптимизировано для минимального взаимодействия с пользователем — установите один раз, используйте ежедневно

Начало работы

Разрешения

Предоставьте эти разрешения при первом запуске:

  • Камера – Необходима для записи видео.
  • Микрофон – Для захвата аудио.
  • Местоположение – Для записи GPS и скорости.
  • Хранилище – (Android 9 и ниже) Для сохранения записей.

Установка телефона

  • Используйте крепление на лобовое стекло или панель приборов со стабильным видом вперед.
  • Держите кабели аккуратно и вне поля зрения.
  • Подключите питание от зарядного устройства, подключенного к зажиганию (ACC), для автоматической работы.

Поведение при питании

  • Зажигание включено (питание подключено): Запись начинается автоматически.
  • Зажигание выключено (питание отключено): Запись безопасно останавливается и начинается загрузка (если настроена). Телефон остается активным для завершения загрузки, затем переходит в спящий режим для экономии энергии.

Главный экран

  • Вверху слева: Дата и время
  • Внизу слева: GPS-координаты и направление
  • Внизу справа: Скорость (миль/ч или км/ч)
  • В центре: Предварительный просмотр камеры и индикатор записи
  • Плавающая кнопка: Нажмите красную, чтобы начать, белую, чтобы остановить ручную запись

Примечание: Если вы видите "EMULATOR – Video recording disabled", вы работаете на виртуальном устройстве. Используйте реальный телефон для записи.

Режимы записи

1. AUTO (Обнаружение ударов)

Записывает непрерывно короткими циклами, сохраняя только последние две минуты. При обнаружении удара (1.5G–3.0G) DashCam сохраняет:

  • Две минуты до удара
  • Десять секунд после
  • Всю телеметрию (GPS + журналы датчиков)

2. PER_TRIP (За поездку)

Начинается при включении зажигания, останавливается при выключении. Сохраняет каждую поездку как один файл с синхронизированными журналами.

3. Циклическая запись по времени (5/10/30 минут)

Разбивает записи на более мелкие ротационные файлы для упрощения загрузки и управления.

4. Ручная

Нажмите красную кнопку, чтобы начать или остановить запись в любое время.

Обзор настроек

Отображение

  • Единицы скорости: миль/ч или км/ч
  • Качество видео: 720p, 1080p (по умолчанию), 4K
  • Формат даты/времени: Региональная настройка
  • Не выключать экран: Предотвращает тайм-аут во время записи

Запись

  • Режим: AUTO, PER_TRIP, 5/10/30 мин
  • Чувствительность к ударам: Низкая (3.0G), Средняя (2.0G по умолчанию), Высокая (1.5G)

Управление хранилищем

Расположение по умолчанию

/Android/data/com.dashcam.app/files/DashCam/

Внешнее хранилище или SD-карта

  1. Перейдите в Настройки → Настройки хранилища.
  2. Выберите ВНЕШНЕЕ.
  3. Нажмите Выбрать пользовательскую папку и выберите путь к вашей SD-карте или USB.

Размеры файлов (приблизительно)

Качество МБ/мин Рекомендуемое использование
720p 50 Длительные поездки, мало места
1080p 80 Сбалансированное качество (по умолчанию)
4K 200 Высокая детализация, большое хранилище

Совет: Журналы телеметрии (.srt, .csv, .gpx) вместе занимают менее 0.2 МБ/мин.

Настройка загрузки

DashCam поддерживает автоматическую загрузку через Wi-Fi в папки SMB/CIFS, Google Drive или OneDrive.

Настройка SMB

  1. Включите загрузку в Настройках.
  2. Установите время загрузки: Вручную, Во время записи или После записи.
  3. Политика сети: Только Wi-Fi (по умолчанию) или Разрешить с оплатой.
  4. Введите данные сервера:
    • Хост: Имя хоста Windows (NetBIOS) или IP-адрес.
    • Папка: Имя общей папки, например, dashcam.
    • Имя пользователя / Пароль: Учетные данные локальной учетной записи Windows.

Разрешение имен хостов (Сети Windows)

DashCam поддерживает разрешение имен NetBIOS, позволяя использовать имена хостов Windows вместо IP-адресов (например, MYPC). Убедитесь, что оба устройства находятся в одной сети Wi-Fi и что Сетевое обнаружение включено в Windows.

Пользователям macOS или Linux может потребоваться подключение с использованием IP-адреса.

Зачем нужна локальная учетная запись Windows

Windows 10+ больше не поддерживает гостевые папки. Создайте выделенного локального пользователя с доступом по паролю:

  1. Win + I → Учетные записи → Семья и другие пользователи.
  2. Выберите Добавить учетную запись → Добавить пользователя без учетной записи Microsoft.
  3. Создайте имя пользователя (например, dashcam_user) и пароль.
  4. Предоставьте папке разрешения Чтение/Запись на вкладках Общий доступ и Безопасность.

Совет: Найдите IP вашего ПК через Командную строку → ipconfig → IPv4-адрес.

Облачная загрузка

DashCam также может загружать напрямую в облачное хранилище.

  • Google Drive: Полностью поддерживается. Авторизуйте свою учетную запись Google один раз в настройках.
  • OneDrive: Полностью поддерживается. Войдите в свою учетную запись Microsoft и выберите целевую папку.
  • Dropbox: Запланировано для будущих версий по запросам пользователей.

Загрузки автоматически повторяются при прерывании, и вы можете выбрать удаление локальных файлов после успешной передачи.

Устранение неполадок

Проблема Возможное решение
Нет видео Предоставьте разрешение камеры; перезапустите приложение.
GPS зависает Переместитесь на улицу для сигнала; убедитесь, что GPS включен.
Удар не обнаружен Отрегулируйте чувствительность; закрепите крепление.
Загрузка не удалась Подтвердите учетные данные; убедитесь в одной сети Wi-Fi; включите порт SMB 445.
Имя хоста не найдено Проверьте обнаружение NetBIOS; используйте IP в качестве запасного варианта.
Хранилище заполнено Снизьте качество видео или включите автоудаление после загрузки.

Советы по производительности

  • Отключите оптимизацию батареи Android для DashCam.
  • Используйте качественное зарядное устройство (2A+).
  • Избегайте перегрева — прямой солнечный свет может вызвать отключение.

Технические примечания

  • Видео: H.264/AVC MP4 @ 30 fps
  • Телеметрия: SRT (GPS/скорость), CSV (данные датчиков 10 Гц), GPX (трек на карте)
  • Буфер ударов: Двухминутный круговой цикл, сохраняет +10с после удара
  • Разрешения: CAMERA, RECORD_AUDIO, ACCESS_FINE_LOCATION, INTERNET, WAKE_LOCK, FOREGROUND_SERVICE
  • Хранение учетных данных: Зашифровано через Android Keystore

Известные ограничения (Beta)

  • Интеграция с Dropbox в ожидании запроса пользователей.
  • Чрезвычайно длительные непрерывные поездки могут требовать большого объема памяти.

Безопасность и ответственность

DashCam помогает водителям — это не замена безопасному вождению. Всегда соблюдайте местные законы относительно записи в автомобиле.
